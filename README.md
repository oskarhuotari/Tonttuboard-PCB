The board includes:
- Holes for the Teensy 4.0
- ICM-40609-D (IMU)
- BMP388 (Barometer)
- W25Q256JVFIQT (Flash Chip)
- LDR holes
- Thermistor holes
- PMW cable holes
- cool graphics
---
Insturctions:
- Buy a teensy 4.0 and order this PCB from JLCPCB
- Solder them together
- Use Tonttulib (or make your own libs for the sensors)
- Fly your project
- Cry
